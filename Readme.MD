###Readable App###

#UDACITY react redux project#
-------
1. open console
2. CD into API-SERVER
3. type npm install
4. type node server.js to launch backend

--------
1. open new console
2. cd into FRONTEND
3. type npm install
4. type npm start
5. open http://localhost:3000 to use the project
